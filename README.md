## How to Contribute
1.  **Technical Way** To contribute, fork the project and update the ontario_townships.geojson file and send a pull request with the changes. You can use Mapbox's nifty http://geojson.io/ project to work with the data in a browser if you are not familar with desktop Geographic Information Systems ("GIS") programs such as [QGIS](http://www.qgis.org/en/site/).
